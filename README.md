## Local development

Requirements: Node.js and npm (recommended via nvm: https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# 1) Clone the repository
git clone <YOUR_GIT_URL>

# 2) Go to project directory
cd <YOUR_PROJECT_NAME>

# 3) Install dependencies
npm i

# 4) Start the dev server
npm run dev
```
